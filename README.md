# ftmag

Food & Travel Magazine — Next.js front end (glass UI, video backdrop).

## Local setup

1. Put `background.mp4` and `logo.png` in the `public/` folder (see `public/PLACE_ASSETS_HERE.txt`).
2. `npm install`
3. `npm run dev` — open [http://localhost:3000](http://localhost:3000)

## Deploy (Vercel)

Connect this repo to Vercel; framework preset **Next.js**. Add environment variables from `.env.example` when you integrate WordPress or a database.

## Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Fonts: Electrolize (nav), Cormorant Garamond (welcome headline)
